# 第19届D2终端技术大会「热·AI」
2025年3月8日，阿里巴巴西溪C区国际总部，第19届D2顺利召开啦！很开心，第19届成为了D2有史以来参会人数最多的一届。本次主题是「热·AI」- 回归初心，热 AI 终端。我们一起来探讨，面对 AI 时代的冲击和机遇，终端人如何破局？
